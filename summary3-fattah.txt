Materi hari ini masih melanjutkan materi command pada git, hari ini kita memraktekan command berikut git checkout -b (branch name), git checkout (branch name) dan git pull origin (branch name), dimana fungsi dari ketiga command tersebut adalah sebagai berikut :

- git checkout -b (branch name) : command ini digunakan untuk membuat branch baru pada repository sekaligus pindah ke branch baru tersebut

- git checkout (branch name) : command ini digunakan untuk pindah ke branch yang telah tersedia

-git pull origin (branch name) : command ini digunakan untuk mengambil data sesuai dengan nama branch ke lokal kita
